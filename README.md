# bdd code-katas
Install dependencies (phpspec)
```bash
composer install -o
```
Run the tests
```bash
phpspec run --format=pretty
```
If you want even more quick feedback loops, use [Gulp](https://github.com/gulpjs/gulp) (you need npm installed)
```bash
npm install 
gulp
```
